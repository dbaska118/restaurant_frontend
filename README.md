## Uruchamianie projektu na systemie Windows
### Wymagania wstępne
- Zainstalowane środowisko **Node.js** (wraz z menedżerem pakietów `npm`)

### Uruchamianie aplikacji
1. Otwórz terminal w katalogu głównym projektu.
2. Zainstaluj wymagane biblioteki.
```
npm install
```
3. Uruchom aplikację.
```
npm run dev
```
